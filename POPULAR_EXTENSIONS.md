# Popular VS Code Extensions to Track

Here are some popular VS Code extensions you can add to your tracker:

## Language Support

- `ms-python.python` - Python
- `golang.go` - Go
- `rust-lang.rust-analyzer` - Rust
- `ms-vscode.cpptools` - C/C++
- `oracle.oracle-java` - Java

## Code Quality

- `esbenp.prettier-vscode` - Prettier Code Formatter
- `dbaeumer.vscode-eslint` - ESLint
- `streetsidesoftware.code-spell-checker` - Code Spell Checker

## Productivity

- `eamodio.gitlens` - GitLens
- `ms-vscode-remote.remote-ssh` - Remote SSH
- `ms-vscode.remote-explorer` - Remote Explorer
- `github.copilot` - GitHub Copilot
- `visualstudioexptteam.vscodeintellicode` - IntelliCode

## Themes

- `pkief.material-icon-theme` - Material Icon Theme
- `github.github-vscode-theme` - GitHub Theme
- `dracula-theme.theme-dracula` - Dracula Official

## Frameworks

- `bradlc.vscode-tailwindcss` - Tailwind CSS IntelliSense
- `vue.volar` - Vue Language Features
- `svelte.svelte-vscode` - Svelte for VS Code

## Testing & Debugging

- `ms-vscode.test-adapter-converter` - Test Adapter Converter
- `hbenl.vscode-test-explorer` - Test Explorer

## Docker & DevOps

- `ms-azuretools.vscode-docker` - Docker
- `ms-kubernetes-tools.vscode-kubernetes-tools` - Kubernetes

Simply copy any extension ID from above and paste it in the "Add Extension" dialog!
